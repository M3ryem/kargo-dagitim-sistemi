# 🚚 Kargo Dağıtım Sistemi


Bu proje, kargo dağıtım süreçlerini yönetmek amacıyla geliştirilmiş bir masaüstü uygulamasıdır.  
Projede kargo ekleme, listeleme,güncelleme ve takip işlemleri yapılabilmektedir.


## ⚙️ Kullanılan Teknolojiler
- C#
- Windows Forms
- .NET Framework


## 🚀 Özellikler
- Kargo ekleme, silme,güncelleme
- Kargoları listeleme
- Dağıtıcı kayıt ekranı
- Dağıtıcı teslimat ekranı
- Kargonun barkod numarasına göre arama 


## 🖥️ Nasıl Çalıştırılır?

 Proje indirildikten sonra '.sln' dosyası Visual Studio ile açılır 

## 🎯 Amaç

 Bu proje, görsel programlama dersi kapsamında geliştirilmiş olup, temel CRUD işlemlerini ve kullanıcı arayüzü geliştirmeyi öğrenmek amacıyla yapılmıştır.
